Key Points

Automatic

Automatic synthesis of program is crucial that it allows us to create of a large-sized benchmark easily, contrasted to manual data creation. We adopted large language model (LLM) to synthesize a program automatically.

Reliability

We need a reliable bug benchmark in order to precisely evaluate bug detectors, such as static analyzers. We adopted program verification technique during the constrained decoding process of LLM to ensure the uniqueness of the bug.

Realistic

Bug benchmark should be realistic; similar enough to manually crafted benchmarks to properly evaluate bug detectors with realistic buggy logic. We checked BugSynth benchmark is similar to the manually crafted benchmark by experiment.
